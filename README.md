# AIMST Barter System 1.0 🔄

An AI-powered, zero-cash sustainability platform designed specifically to optimize resource sharing and match trades seamlessly across the AIMST University campus.

---

## 🚀 Key Architectural Features

* **🤖 Autonomous AI Matching Agent Engine:** Utilizes an optimized keyword-parsing heuristic algorithm that scans the campus database live. The moment a bidirectional trade loop occurs (User A has what User B wants AND User B has what User A wants), the system dynamically pushes a live "Trade Loop Discovered" card to the UI layout.
* **🔑 Secure Student Whitelist Registry Layer:** Protects campus security by requiring verified profile whitelist tokens before allowing any media or listing uploads. Core data attributes like Student IDs are encrypted or hidden from unauthorized public access channels.
* **💾 Local Persistence Engine:** Architected using absolute directory mappings to handle fail-safe JSON tracking registries (`marketplace_data.json` and `users_registry.json`) across both local workspaces and cloud server architectures.

---

## 🛠️ Technical Stack

* **Backend Engine:** Python 3.10 / Flask Micro-framework
* **Dynamic Rendering:** Jinja2 Template Engine (Strict server-side security context wrappers)
* **Image Processing Pipeline:** Pillow (PIL) for asynchronous image optimization, thumbnail scaling, and format normalization
* **Version Control & Deployment:** Git CLI / Fully deployed to a live PythonAnywhere production server instance
